# Alves Construction Website

## Description:

This website showcases the services offered by Alves Construction, including residential and commercial construction, renovations, and more.

## Features

###### Main Features:

- Auto-typing banner for catchy slogans
- Responsive design for Mobile, Tablet and Computers
- Comment section for user feedback
- Background color wave effect
- Logo with spin effect
- Contact Form
- Links to social media
- Videos & Pictures
-

###### Navigation:

- Navigation bar with icons for easy access to different pages
- Quick Links Bar at bottom of page (All Pages/ Links)

## Technologies Used:

- HTML
- CSS
- JavaScript
- Typed.js library (auto-typing effect)
- Font Awesome
